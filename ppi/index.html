Resumo

A ideia inicial do projeto é criar um software educacional, que ensine intuitivamente o que é e para que serve cada item do seu computador, permitindo assim que o usuário tenha uma facilidade maior no uso da tecnologia.

Introdução

Objetivo Geral

Desenvolver o projeto de um software educacional que auxilie e promova a inclusão digital de pessoas que nunca usaram ou que tenham dificuldade em utlizar o computador.
